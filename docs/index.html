Desarrollador Web

Como desarrollador me esfuerzo para ampliar mis conocimientos y experiencia para poder afrontar proyectos cada vez más grandes. Disfruto el proceso de diseño de aplicacion y de bases de datos, y acojo el concepto de lo que significa ser un desarrollador full-stack y la necesidad de ser competente en todas las tecnologías utilizadas para hacer aplicaciones web complejas. También, aunque me considero principalmente un desarrollador, debido a mi historia como administrador de sistemas y sentida preferencia por GNU/Linux como sistema operativo diario y para entornos de producción; tengo experiencia hosteando sitios web en servidores remotos usando GNU/Linux.
